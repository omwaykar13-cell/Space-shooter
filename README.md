Space Shooter
A simple browser-based space shooter game built with HTML, CSS, and JavaScript. Open index.html in a browser to play locally.

How to play
Use the arrow keys or WASD to move the ship.

Press Space to shoot.

Destroy enemies and avoid collisions to score points.
press s for more spaceship skin

Files
index.html — game entry point.

styles.css — game styling.

main.js — game logic.

assets/ — images and audio used by the game.

Run locally
Open index.html directly in your browser: double-click the file or drag it into the browser.
